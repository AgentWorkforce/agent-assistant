# examples

Status: reference-only directory, not a production runtime package.

## Purpose

This directory is intended to hold reference examples that show how products should adopt the SDK.

Consumers should expect this directory to contain:

- skeletal assistant examples
- example composition patterns
- migration-oriented reference setups

## Expected Consumer Role

Consumers should use these examples for guidance, not as a production dependency for product logic.

Recommended posture:
- read from here
- copy patterns from here
- do not depend on this directory as a required runtime package in production product code

## What Stays Outside

- production product code
- private adapters
- hosted cloud examples that require non-OSS infrastructure
